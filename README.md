# Olin Dashboard

A modern, responsive dashboard UI for the Olin platform.

## Features

- Clean and modern design
- Responsive layout
- Interactive sidebar navigation
- Dashboard with multiple tool sections
- Matter management interface

## Structure

- `index.html` - Main HTML structure
- `style.css` - Stylesheet with CSS variables and responsive design
- `images/` - Image assets (logos, avatars, icons)

## Technologies

- HTML5
- CSS3 (with CSS Variables)
- Google Material Icons
- Google Fonts (Inter)

## Setup

Simply open `index.html` in a web browser to view the dashboard.
